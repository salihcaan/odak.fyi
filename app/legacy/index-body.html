FAQ

Quick answers.

The things people ask before they try it.

Will this run on my Mac?

macOS 26 Tahoe or later, Apple Silicon only. The Liquid Glass UI uses macOS 26 APIs that don't backport, and Tahoe dropped Intel — so neither does Odak.

I already use Raycast — why this?

Raycast knows apps and commands; Odak knows projects (which editors have them open, what each one wants you to do next). Use them side by side.

Which editors does it support?

Cursor, VS Code, IntelliJ IDEA, and GoLand out of the box. Xcode and anything else via Settings — pick the .app and Odak grabs the bundle ID for you.

What goes in .odak vs ~/.odak/config.yaml?

The .odak file in a project is tiny — just the IDE to use and a few variables. Your actions live globally in ~/.odak/config.yaml; write them once, they apply everywhere. Read the full reference →

What does Odak send back?

By default, nothing — projects, queries, and settings never leave your Mac. If you opt in, Odak shares anonymous events and crash reports (no email, no IP, no project names, no window titles). Full details →
